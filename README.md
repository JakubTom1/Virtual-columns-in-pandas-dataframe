# Virtual-columns-in-pandas-dataframe
A lightweight Python utility to safely append a calculated virtual column to a `pandas` DataFrame based on a string mathematical expression. It features strict validation for column names, existing variables, and allowed arithmetic operations (`+`, `-`, `*`).

## Requirements

- Python 3.7+
- `pandas`
- `pytest` (for running tests)

You can install the required packages using pip:

```bash
pip install pandas pytest
